# UI Components

A scalable and easily extensible **UI Kit built with React and TypeScript**.

Designed for rapid prototyping and as a flexible foundation that can be adapted to specific product needs.

---

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Storybook
- Vitest + Testing Library
- Vite

---

## Characteristics

- Fully typed with strict TypeScript support
- Accessibility-aware (a11y-first approach)
- Easily extensible and scalable architecture
- Reusable and composable components
- Controlled and uncontrolled patterns
- Covered with interaction tests
- Fully documented in Storybook
- Design-system oriented structure (shared tokens, consistent variants, predictable API)
